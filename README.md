# 🎬 Movie Recommendation System

A Machine Learning based movie recommender built with Streamlit.

## 🚀 Live Demo
👉 https://movierecommendationsystem-3mq6vwdufyy52tu8eutmdf.streamlit.app/

## Features
- Recommends 5 similar movies
- Displays movie posters using TMDB API
- Built with Python, Pandas, and Streamlit
